# 🎯 FIFA Project - Web Applications

A collection of interactive web applications built with HTML, CSS, and JavaScript.

## 📱 Applications Included

### 1. Football Game
- Interactive 2D football/soccer game
- Player vs AI opponent
- Real-time scoring and timer
- Ball physics and collision detection

**Play:** Open `index.html`

### 2. Joke Generator
- Fetches random jokes from external API
- Multiple joke categories (General, Programming, Knock-Knock)
- Share jokes via Web Share API or clipboard
- Joke history with recent jokes tracking
- Fully responsive design

**Play:** Open `joke-generator.html` (after updating)

### 3. To-Do List App
- Add, edit, and delete tasks
- Priority levels for each task
- Smart filtering (All, Active, Completed, High Priority)
- Live statistics and progress tracking
- Local storage persistence
- Export tasks to CSV

**Use:** Open `todo.html`

## 📝 Usage Tips

### Football Game
- Use Arrow Keys or WASD to move
- Press Space to kick the ball
- Press R to reset the game

### Joke Generator
- Click "Get Joke" to fetch a new joke
- Select joke type from dropdown
- Click "Share Joke" to copy to clipboard
- Click recent jokes to view them again

### To-Do List
- Enter task and select priority
- Press Enter or click "Add Task"
- Click checkboxes to mark complete
- Use filters to organize view
- Export tasks as CSV file
